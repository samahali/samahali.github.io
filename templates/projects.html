Backend Developer – AYEN


(Dec. 2020 – Jul. 2022)
Project: Property Inspection Platform
A web application designed to streamline property inspections, allowing users to manage property Details, schedule inspections, and generate reports.
Technologies Used:
  • Backend: Django Templates, Django REST Framework (DRF 3.1), Python 3.9.
  • Database: MySQL.
  • Version Control & Containerization: Git, Docker.
  • Third-Party Integrations: Amazon Payment Services, Apple Pay, Qoyod, SendGrid, Bitly, Google Sheets (gspread), Zapier
  • Monitoring & CI/CD Tools: Sentry, SonarQube, Slack, Ngrok, GitLab, Jira
Role & Responsibilities:
  • Developed and maintained server-side components using Django and DRF.
  • Integrated payment gateways (Amazon Payment Services, Apple Pay) and financial management systems (Qoyod).
  • Implemented email communication via SendGrid, link shortening with Bitly, and automated reporting using Google Sheets (gspread).
  • Worked closely with the Angular front-end team to ensure seamless API integration.
  • Used Docker for containerization and implemented CI/CD pipelines using GitLab, Sentry, and SonarQube for monitoring and testing.

Backend Developer – Crypto Payment Application (Paymob)


(Nov. 2022 – Nov. 2024)
Developed a crypto payment application enabling secure transactions, withdrawals, and swaps across major blockchain networks. Integrated real-time transaction monitoring and QR code generation for seamless payments.
Technologies Used:
  • Backend: Django 4.1, Python 3.11.
  • Blockchain: Web3 (Ethereum), Solana Package (Solana)
  • Real-Time Communication: Django Channels (WebSocket)
  • Database: MariaDB
  • DevOps & Tools: Git, Docker, GitLab CI/CD, Sentry, SonarQube, Ngrok, Jira
Role & Responsibilities:
  • Developed core backend features, including payments, withdrawals, and swaps.
  • Integrated Fireblocks and Circle for secure cryptocurrency management.
  • Built a real-time WebSocket system (Django Channels) for blockchain transaction monitoring.
  • Collaborated with blockchain developers to ensure smooth Ethereum & Solana integration.
Challenges & Solutions:
  • Multi-Blockchain Integration: Used Web3 for Ethereum and Solana Package for Solana.
  • Real-Time Transaction Monitoring: Implemented async WebSockets to track payments.
  • Refund Handling: Developed mechanisms to ensure correct user refunds on failed transactions.
